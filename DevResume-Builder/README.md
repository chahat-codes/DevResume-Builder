# DevResume Builder 🚀

A sleek, framework-less resume generator designed for developers. Build your professional resume in real-time and export it as a clean PDF.

![DevResume Preview](https://via.placeholder.com/800x400?text=DevResume+Builder+Preview)

## ✨ Features

- **Live Preview:** See your resume update instantly as you type.
- **Dual Themes:** Switch between **Classic** (professional) and **Modern** (stylish) layouts.
- **Local Persistence:** Your data is automatically saved to your browser's local storage.
- **PDF Export:** Print-optimized styles ensure your resume looks perfect on paper or as a PDF.
- **Zero Dependencies:** Built using only Vanilla HTML, CSS, and JavaScript.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables), Flexbox, and Grid.
- **JavaScript (ES6+):** DOM manipulation and LocalStorage API.
