# Contributing to DevResume Builder 🤝

Thank you for your interest in contributing to **DevResume Builder**! We welcome contributions from everyone, especially first-time open-source contributors.

## 🚀 Getting Started

1. **Fork the Repository:** Click the "Fork" button at the top right of the repo page.
2. **Clone your Fork:**
   ```bash
   git clone https://github.com/your-username/DevResume-Builder.git
   ```
3. **Create a Branch:**
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your Changes:** Add your code, fix bugs, or improve documentation.
5. **Commit your Changes:**
   ```bash
   git commit -m "Add: Descriptive message about your change"
   ```
6. **Push to GitHub:**
   ```bash
   git push origin feature/your-feature-name
   ```
7. **Open a Pull Request (PR):** Go to the original repository and click "New Pull Request".

## 🛠️ Contribution Areas

- **Themes:** Create a new CSS file in the `themes/` directory and update `script.js` to support it.
- **Features:** Add support for social icons, profile pictures, or custom sections.
- **UI/UX:** Improve the form layout or preview responsiveness.
- **Bug Fixes:** Check the "Issues" tab for reported bugs.

## 📜 Code of Conduct

- Be respectful and inclusive.
- Provide clear and concise PR descriptions.
- Ensure your code is well-commented and clean.

## 🌟 GSSoC Contributors

If you are contributing via GSSoC:
- Always comment on an issue before starting work.
- Wait for the issue to be assigned to you.
- Include your GSSoC ID in the PR description.

Happy Coding! 💻
